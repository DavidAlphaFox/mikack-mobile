# mikack-mobile

Mikack 手机客户端

## 基本说明

本项目是由 **Flutter** 框架构建的 Mikack GUI 客户端，主要面向移动设备设计。

## 当前状态

在未来很长时间内可能都不支持 IOS，但应该不是永久性的。即便是 Android 平台目前也处于开发阶段，会遇到功能不完善或 BUG 导致的体验问题。

**待基础功能完善，将会在 Release 页面上传 apk 文件。**
